# マイ練習 (MyRenshuu)

A Japanese practice application built in Java using Swing.

# Features
- Vocabulary Flashcards
- Writing Practice (English → Japanese)
- Matching Game
- Fill in the Blank (sentence-based)

# Core Features and Implementations
- Java
- Swing GUI
- File I/O
- Object-Oriented Programming
## Project Structure
- src/ → Java source code
- vocabulary.txt → vocabulary data
- vocabulary_fill_blank.txt → sentence practice data
